## iCloud API playground

This folder contains experimental scripts for working with iCloud via `pyicloud`.

### Purpose
- test iCloud authentication
- manage trusted sessions (2FA, cookies)
- experiment with photo downloads via API

### Notes
- iCloud cookies are stored **outside the repository**
- no credentials or session data are committed
- scripts are numbered to reflect execution order


## iCloud API — песочница

В этом каталоге находятся экспериментальные скрипты для работы с iCloud через `pyicloud`.

### Назначение
- проверка авторизации iCloud
- управление доверенными сессиями (2FA, cookies)
- эксперименты со скачиванием фото через API

### Важно
- cookies iCloud хранятся **вне репозитория**
- логины и сессии в git не коммитятся
- файлы пронумерованы по порядку выполнения
